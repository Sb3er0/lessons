Иконки

Иконки - векторные, простые, лаконичные, но при этом очень выразительные. С их помощью легко привлечь внимание посетителя к какому-либо разделу сайта или донести до него важную информацию. В большинстве случаев векторные изображения.

Иконки для навигации. Из названия понятно, что эти иконки нужны для того, чтобы людям было проще ориентироваться на сайте.

Информационные иконки. Иконки — это не обязательно ссылки. С их помощью можно показать, какие услуги вы оказываете, рассказать про ключевые ценности компании или преимущества вашего продукта.

Иконки-призывы к действию. Ими надо пользоваться с осторожностью, потому что люди не всегда правильно понимают их смысл. Жирную стрелочку в круге можно трактовать и как «Скачать», и как «Вниз». У значка «+» тоже несколько распространенных значений — и «Добавить в корзину», и «Узнать больше информации». Если вы используете подобные символы в качестве кнопок, советуем добавлять к ним короткие подписи — для большей ясности.

Выводятся через:
html ➔ IMG
html ➔ SVG(кодом) p.s. наведите на иконки соц. сетей
CSS ➔ ::before/::after
Формат: SVG

Элементы дизайна

Элементы дизайна могут быть как векторными, так и растровыми изображениями. Если есть возможность, то можно вывести векторное изображением через «html ➔ SVG(кодом)» потому что так можно менять цвет данного элемента(если же он является векторным.

Выводятся через:
html ➔ SVG(кодом, только если векторное изображение)
CSS ➔ ::before/::after.
CSS ➔ background/background-image.
Формат PNG, SVG, WEBP.

Контентные изображения

Контентные изображения – изображения, являющиеся контентом сайта. Обычно это растровые изображения.

Так же есть изображение заднего фона «background» – изображение является задним фоном.

Выводятся через:
html ➔ IMG
CSS ➔ background/background-image.
Формат JPG, PNG, WEBP.
